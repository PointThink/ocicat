Ocicat

Ocicat is a framework for making 2D games in C# and other .NET languages.

Ocicat provides an easy to use API for low-level features like graphics, audio etc. Ocicat is NOT a game engine. It doesn't provide any fancy graphical editors. It's all raw C#.

Examples

All examples are avialable in the GitHub repo. Please note that not all features might be in the examples.
